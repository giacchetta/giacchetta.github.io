  Dirigí el diseño e implementación de una arquitectura de infraestructura integral en GCP, empleando un enfoque orientado a SaaS con múltiples entornos mediante módulos de Terraform. Esto incluyó la configuración de un balanceador de carga para servir contenido estático desde un bucket de backend y contenido dinámico vía funciones serverless conectadas a una API REST, uniendo los servicios de frontend y backend.
  Mis responsabilidades se extendieron a colaborar en el despliegue del frontend (TypeScript) y desarrollar e implementar de forma independiente todos los servicios de backend (NodeJS). Este robusto backend constaba de más de 60 funciones serverless interconectadas con servidores SQL y NoSQL en una arquitectura full-mesh, complementada por una cola de mensajería RabbitMQ. Para el monitoreo, aproveché el soporte nativo de Prometheus en la nube, asegurando que todas las funciones cloud estuvieran monitoreadas con integración de SDK y alertas configuradas para el turno de guardia. A lo largo de estas iniciativas, lideré y colaboré activamente utilizando metodologías ágiles.
